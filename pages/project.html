TaskFlow

TaskFlow is a full-stack project designed to manage tasks, track status changes, and streamline small team workflows.

I built the backend using Python, exposed the functionality through a clean REST API, and created a simple but responsive front-end interface for creating, filtering, and updating tickets. This project helped me practice database design, API structuring, state management, and front-end rendering in a real-world style application.

Medical Inventory

MedInventory was built to simplify daily medication management by providing a clean UI for adding medications, storing dosage details, and setting reminder times. It includes a searchable medication list, a structured local database, and a notification-ready design that can be extended to mobile platforms.

This project allowed me to explore CRUD operations, data modeling, user interface flow, and practical problem-solving for healthcare-related tools.
